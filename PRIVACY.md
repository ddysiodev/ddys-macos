# 隐私说明

DDYS macOS 以本地优先为原则。

## 本地保存的数据

应用会在当前 Mac 保存以下数据：

- API Base、站点 Base、主题、缓存、刷新间隔、通知等设置
- 收藏、稍后看、观看历史
- 影片备注
- 关键词订阅
- API 响应缓存
- 最近一次诊断结果

这些数据不会被应用主动上传。

## 网络请求

应用会在以下场景请求用户配置的 DDYS API 或站点地址：

- 搜索、发现页刷新、日历加载
- 打开影片详情、资源和相关内容
- 检查关键词订阅
- 运行诊断
- 用户点击打开站点或资源链接

## macOS 权限

桌面端使用 Tauri v2 能力：

- WKWebView 显示应用界面
- 菜单栏托盘用于显示或退出应用
- 通知能力用于订阅提醒
- 打开外链能力用于跳转影片页面或资源链接
- 用户选择的 JSON 文件用于导入本地数据

## 导入导出

导出的 JSON 文件包含本地设置、收藏、稍后看、历史、备注和订阅。请只在可信设备之间传输。
